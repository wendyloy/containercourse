# NUS-ISS Containers for Deployment and Scaling 

This is for the submission of workshops for NUS-ISS Containers for Deployment and Scaling course. 

## Workshop 01
- Build a Docker image 
- Deploy 2 containers inside Docker network 

## Workshop 02
- Build a container image 
- Deploy the image to a Kubernetes cluster 
- Make the deployment accessible from the outside of the cluster by exposing it as a load balancer 

## Workshop 03
- Deploy an application and its associated database 
- Externalized all application configurations  
- Create a volume for the database 
- Create an Ingress resource to allow the application to be accessible from outside of the cluster

## Workshop 04
- Deploy an object store cluster
